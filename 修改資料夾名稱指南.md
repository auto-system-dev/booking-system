# 修改資料夾名稱指南

## ✅ 可以修改資料夾名稱

**是的，可以修改！** 將中文資料夾名稱改為英文可以避免編碼問題。

## ⚠️ 注意事項

### 修改前需要確認：

1. **Git 倉庫不受影響**
   - Git 追蹤的是檔案內容，不是資料夾路徑
   - 修改資料夾名稱不會影響 Git 倉庫
   - 但需要重新設定工作目錄

2. **需要重新打開 Cursor**
   - 修改後需要關閉並重新打開 Cursor
   - 或重新設定工作目錄

3. **確保沒有程式正在使用**
   - 關閉所有終端機
   - 確保沒有程式正在讀寫這個資料夾

---

## 🔧 修改步驟

### 步驟 1：關閉 Cursor 和所有終端機

1. 關閉 Cursor
2. 關閉所有 PowerShell 或終端機視窗
3. 確保沒有程式正在使用這個資料夾

### 步驟 2：修改資料夾名稱

#### 方法 A：使用檔案總管（推薦）

1. 打開檔案總管
2. 導航到 `C:\Users\user\Desktop`
3. 找到 `程式開發` 資料夾
4. 右鍵點擊 → **重新命名**
5. 改為 `Programming` 或 `booking-system`（建議使用專案名稱）
6. 按 Enter 確認

#### 方法 B：使用 PowerShell（如果檔案總管無法修改）

1. 打開 PowerShell（以系統管理員身分執行）
2. 執行：
```powershell
# 先設定編碼
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8

# 修改資料夾名稱
Rename-Item -Path "C:\Users\user\Desktop\程式開發" -NewName "Programming"
```

### 步驟 3：重新打開 Cursor

1. 打開 Cursor
2. 選擇 **File** → **Open Folder**
3. 選擇新的資料夾路徑：`C:\Users\user\Desktop\Programming`
4. 或直接拖放資料夾到 Cursor

### 步驟 4：驗證 Git 是否正常

在新的資料夾中打開終端機，執行：
```bash
git status
```

應該可以看到正常的 Git 狀態，不會有錯誤。

## 📋 建議的資料夾名稱

### 選項 1：使用專案名稱（推薦）
```
booking-system
```
- 清楚明確
- 與 GitHub 倉庫名稱一致
- 專業

### 選項 2：使用描述性名稱
```
Programming
```
- 簡單明瞭
- 通用名稱

### 選項 3：使用更具體的名稱
```
booking-system-dev
hotel-booking-system
```
- 更具描述性
- 如果有多個專案可以區分

## 🆘 如果遇到問題

### 問題 1：無法重新命名（資料夾正在使用）

**解決方案：**
1. 關閉 Cursor
2. 關閉所有終端機
3. 檢查工作管理員，結束相關程序
4. 再試一次

### 問題 2：Git 找不到倉庫

**解決方案：**
1. 確認在新的資料夾中
2. 執行 `git status` 確認
3. 如果還是不行，可能需要重新初始化（但這會失去歷史記錄，不推薦）

### 問題 3：Cursor 找不到檔案

**解決方案：**
1. 關閉 Cursor
2. 重新打開 Cursor
3. 使用 **File** → **Open Folder** 選擇新資料夾
