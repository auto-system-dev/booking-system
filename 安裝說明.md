# 正確的安裝步驟

## ⚠️ 重要提醒

**不要在 `C:\Program Files\nodejs` 目錄下執行安裝！**

應該在**專案目錄**（`C:\Users\user\Desktop\程式開發`）下執行。

## ✅ 正確的安裝步驟

### 步驟 1：切換到專案目錄

在 PowerShell 或 CMD 中執行：

```powershell
cd "C:\Users\user\Desktop\程式開發"
```

### 步驟 2：執行安裝

```powershell
& "C:\Program Files\nodejs\npm.cmd" install
```

或使用批次檔：
- 雙擊 `快速啟動.bat`（會自動安裝依賴）

## 📋 安裝完成後

您應該看到：
```
up to date, audited XXX packages in Xs
```

這表示所有套件（包括 sqlite3）都已安裝完成。

## 🚀 下一步

1. **重新啟動伺服器**：
   ```powershell
   & "C:\Program Files\nodejs\node.exe" server.js
   ```

2. **確認資料庫初始化**：
   啟動時應該看到：
   ```
   ✅ 已連接到 SQLite 資料庫
   ✅ 資料表已準備就緒
   ```

3. **測試訂房功能**：
   - 提交一筆訂房
   - 資料會自動儲存到 `bookings.db`

## 💡 提示

- 所有 `npm` 命令都應該在**專案目錄**下執行
- 使用批次檔可以避免路徑問題
- 資料庫檔案會自動建立在專案目錄中
